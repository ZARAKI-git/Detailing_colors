## Overview

A self-contained, dependency-free marketing page for a car detailing and ceramic coating
business. Everything — markup, styling, and interactivity — lives in a single HTML file,
making it trivial to host anywhere with zero build step.

## Features

- **Single-file build** — no dependencies, no bundler, no install.
- **Cinematic visual design** — premium, studio-style presentation for the detailing brand.
- **Fully responsive** — works across desktop and mobile.
- **Instant hosting** — drop it on any static host or open it directly in a browser.

## Usage

Clone the repo and open the page in a browser:

```bash
git clone https://github.com/ZARAKI-git/Detailing_colors.git
cd Detailing_colors
open detailing-colors-cinematic_7.html   # or just double-click it
```

To serve it locally over HTTP:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000/detailing-colors-cinematic_7.html
```

## Deployment

Being a single static file, it can be deployed to any static host — GitHub Pages, Netlify,
Vercel, Cloudflare Pages, or a simple object store. No build configuration is required.
